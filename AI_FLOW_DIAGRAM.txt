7. SOFTMAX ACTIVATION
================================================================================
   
   Logits
        |
        v
   ┌─────────────────────────────────────┐
   │  Softmax Function                   │
   │                                     │
   │  P(i) = exp(logit_i) / Σ exp(logit_j) │
   │                                     │
   │  Example:                           │
   │  logit = [2.5, 1.2, 0.3, ...]       │
   │  exp   = [12.18, 3.32, 1.35, ...]   │
   │  sum   = 16.85                      │
   │  prob  = [0.72, 0.20, 0.08, ...]    │
   └─────────────────────────────────────┘
        |
        v
   Probabilities [0.85, 0.10, 0.03, 0.01, 0.00, 0.00, 0.00]
   (Sum = 1.0)

================================================================================
                        MATHEMATICAL OPERATIONS
================================================================================

1. Convolution:
   output = Σ(input * kernel) + bias

2. Self-Attention:
   Attention(Q,K,V) = softmax(QK^T / √d_k) * V

3. Softmax:
   P(i) = exp(x_i) / Σ exp(x_j)

4. ReLU:
   f(x) = max(0, x)

5. Layer Normalization:
   y = (x - μ) / √(σ² + ε)
