### 2. 🎤 Live Voice Recording
**File**: `live_recorder.py`

**Features**:
- Record from microphone
- Custom duration (default 5s)
- Continuous recording mode
- List audio devices
- Device selection
- Auto-save recordings
- Real-time emotion detection

**Commands**:
```bash
# Basic recording
python live_recorder.py

# Custom duration
python live_recorder.py --duration 10

# Continuous mode
python live_recorder.py --continuous

# List devices
python live_recorder.py --list-devices

# Use specific device
python live_recorder.py --device 1

# Don't save
python live_recorder.py --no-save
```
